# Free-On-Campus-Resources

1. Clone Repo/Project
```
git clone <repo-url>
cd Free-On-Campus-Resources
```

2. Create Python Virtual Environment
Create & Source
```
python3 -m venv venv
```

3. Install Dependencies
```
pip install flask
pip install flask-cors
pip install werkzeug
```

4. Run Application
Go into /Databases/ directory and enter:
```
python3 app.py
```
Ctrl+Left Click on Localhost address: "Running on http://127.0.0.1:5000"
